Easy to use, light weight, image and content slider for Gutenberg editor.

== Description ==

[youtube https://www.youtube.com/watch?v=6z3DGUz8WsE]

Easy to use, light weight, image and content slider for Gutenberg editor.

= Light weight =

4KB javascript file.

= Image slider =

Background image with parallax effect.

= Content slider =

Use any Gutenberg blocks as content and design it your way.

= Gradient / color overlay =

Add Gradient or color overlay over image.

== Slider features ==

* Slide and fade effect
* Parallax background image
* 20 slide animations
* Works with WordPress Gutenberg
* Customize gradient or solid color overlay
* Infinite loop
* Dots or titles slide indicator
* Custom slider indicator colors
* Slider animation & speed controls

== Installation ==

1. Visit **Plugins > Add New**.
2. Search for **super block slider**.
3. Install and activate the super block slider plugin.

= Manual installation =

1. Upload the entire `super-block-slider` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate super block slider.

= After activation =

While in Page/Post edit mode simply add/search for **super block slider**.
